# SPRING PLUS
## 개선 및 성능 최적화 

이 프로젝트는 대용량 데이터를 처리하고, 검색 성능 최적화와 트랜잭션 관리, AWS 서비스를 활용한 배포 등을 포함하여 실무에서 발생할 수 있는 여러 문제를 해결하는 과정을 다룹니다.

---

## 기술 스택 (Tech Stack)
- Java: 17
- Spring Boot: 3.3.3
- MySQL: 8.4
- AWS
  - EC2
  - RDS
  - S3
- Spring Security
- Spring Data JPA
- QueryDSL
- JWT (JSON Web Token)
- IntelliJ version: 25.2.2
- API Test: Postman
